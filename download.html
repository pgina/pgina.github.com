3.0.1 BETA

Posted by Nate Yocom

This is the 3.0.1 BETA installer for x86 and x64 Windows.

Supported platforms:

  • Windows XP SP3
  • Windows 2003 SP2
  • Windows Vista
  • Windows 7
  • Windows 2008
  • Windows 2008 R2
  • Windows 8 Developer Preview Release*

Included plugins:

  • LDAP
  • Local SAM
  • Single User
  • MySQL Logging
  • Session Limit

* Note that this is true for the Developer Preview released mid-september 2011. It remains to be seen whether changes in the as-of-yet-unreleased Windows 8 affect pGina compatibility.
